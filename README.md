Welcom to atmos(the weather forecast app)

atmos shows weather forecast of 5 days for Bengaluru, IN

Below are the how to's .<br>


## How to's
  #Clone: <br>
    - Git Clone<br>
    - yarn Install <br>
  <br>
  #Run:<br>
    - yarn start
  <br><br>
  #Build:<br>
    - yarn build
  <br><br>
  #Unit Tests:<br>
    - yarn test:unit<br>
    <br>
  #JS Lint:<br>
    - yarn test:lint:js<br>
    <br>
  #CSS Lint:<br>
    - yarn test:lint:css<br>
  <br>
